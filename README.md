# Reflector

### Roadmap
- [x] Copy between text channels
- [x] Impersonate user with nickname and profile pic
- [x] Copy attachments such as text and image files
- [x] Copy between guilds, creating text-channels where they don't exist
- [ ] Copy history in text-channels
- [ ] Rate limit management
- [ ] Archive channel deletions to a archive category and make it private
- [ ] Copy guild logo
#### Extended scope
- [ ] Copy roles
- [ ] Copy channel settings and permissions
- [ ] Copy channels structure with categories
- [ ] Copy soundboard audio files and configuration
- [ ] Copy stickers
- [ ] Assign roles to users in mirror guild which they have in source guild (needs state)
